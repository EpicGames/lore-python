# Code of Conduct

The Lore Python SDK follows the Lore project Code of Conduct, based on the
[Contributor Covenant 3.0](https://www.contributor-covenant.org/version/3/0/).
We are committed to a welcoming, safe, and equitable community.

To report a violation, email **lore-moderation@epicgames.com**. Reports are
reviewed by Community Moderators and handled confidentially.

The full Code of Conduct — including encouraged behaviors, restricted behaviors,
and the enforcement ladder — is published in the Lore repository:

→ [Lore CODE_OF_CONDUCT.md](https://github.com/EpicGames/lore/blob/main/CODE_OF_CONDUCT.md)
